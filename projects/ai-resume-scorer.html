AI Resume Scorer

Инструмент для оценки резюме и выдачи рекомендаций с помощью LLM.

LLM Python Streamlit / Gradio

Идея

Ускорить отбор кандидатов: резюме прогоняется через LLM, которая выставляет оценку по критериям (структура, релевантность, ключевые навыки) и предлагает конкретные улучшения.

Как работает

  • Пользователь загружает резюме (или вставляет текст);
  • LLM анализирует документ по заданному промпту;
  • На выходе — числовая оценка и список улучшений по пунктам.

Стек

  • Python;
  • OpenAI API;
  • Streamlit или Gradio для интерфейса.
