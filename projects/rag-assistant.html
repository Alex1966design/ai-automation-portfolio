AI-ассистент «Моё Дело»

Корпоративный RAG-ассистент по налоговой реформе 2026 на базе внутренних документов.

RAG Qdrant OpenAI API Gradio Python

Цель проекта

Сократить время поиска ответов по новой налоговой реформе для сотрудников и клиентов компании «Моё Дело». Сделать ассистента, который отвечает строго по внутренней базе знаний, без "выдумок" модели.

Что делает ассистент

  • Индексирует документы компании (PDF, DOCX, регламенты) в Qdrant;
  • Получает вопрос пользователя и ищет релевантные фрагменты;
  • Передаёт найденный контекст в LLM (OpenAI) с жёстким системным промптом;
  • Возвращает развернутый ответ с отсылкой к источникам.

Технологический стек

  • Python (OpenAI SDK, qdrant-client, python-dotenv);
  • Qdrant — хранение векторных представлений документов;
  • Модель embeddings: text-embedding-3-small;
  • LLM: gpt-4.1-mini (по состоянию на разработку);
  • Gradio — интерфейс для сотрудников/клиента.

Результат

Время получения ответа на сложные вопросы по реформе сократилось с 15–20 минут (ручной поиск) до ~15 секунд. Ассистент возвращает только то, что реально есть в документах, и явно говорит, если информации недостаточно.
