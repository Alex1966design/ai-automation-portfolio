Summary Bot

Автоматическая генерация обзоров переписок в Telegram с использованием LLM и n8n.

n8n Telegram Bot API OpenAI Webhooks

Задача

Автоматически подводить итоги обсуждений в рабочих чатах: собирать сообщения за период и генерировать краткое резюме, которое уходит в отдельный канал или чат.

Архитектура решения

  • n8n Webhook принимает входящие сообщения из Telegram;
  • Сообщения сохраняются и агрегируются за выбранный период;
  • По расписанию (cron) вызывается узел LLM (OpenAI) для генерации summary;
  • Готовый обзор отправляется обратно в Telegram-канал/чат.

Технологии

  • n8n (webhook, cron, HTTP Request, OpenAI node);
  • Telegram Bot API / токен бота;
  • OpenAI API для генерации summary;
  • Опционально — Google Sheets / БД для хранения истории.

Что это даёт бизнесу

Руководители и участники команд получают короткий обзор без необходимости перечитывать весь чат. Это удобно для созвонов, статус-апдейтов и передачи задач.
