PilotAccessBot

Демо-бот с авторизацией и хранением профилей пользователей в SQLite.

Telegram Bot Python SQLite

Функциональность

  • /start — регистрация пользователя в базе;
  • /profile — вывод сохранённого профиля;
  • Хранение telegram_id, username, имени и ФИО.

Техническая реализация

  • Python + python-telegram-bot;
  • SQLite для хранения пользователей;
  • Простая архитектура, готовая к расширению под роли/доступы.
